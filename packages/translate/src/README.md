# 翻译模块

支持中文、英文

## export api

| 属性               | 说明           | 类型               |
| ------------------ | -------------- | ------------------ |
| unStandard         | 长文本翻译     | Object             |
| SupportedLangauges | 支持的语言     | Object             |
| getCurrentLanguage | 获取本地语言   | Function           |
| localLang          | 本地语言       | string: 'zh'、'en' |
| translations       | 短文案翻译字典 | Object             |
